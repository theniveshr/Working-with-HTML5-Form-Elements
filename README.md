# Working-with-HTML5-Form-Elements

Built a site for the given application title that conforms to HTML5 standards and has the following elements: doctype, html, meta (in the revised html5 format), title, body, head, div, heading, form, input(text, password, email, tel, date, number, radio, checkbox, url, search, file upload, submit, reset), textarea, select, progress, fieldset, label, and html comment.
